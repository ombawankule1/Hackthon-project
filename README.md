# Smart Grievance Management System

## Problem
Citizens face delays and lack of transparency while submitting complaints to authorities.

## Solution
We built a web platform where users can submit complaints with location and track their status in real-time.

## Features
- User complaint submission
- Location-based reporting
- Status tracking dashboard
- Admin status update system

## Tech Stack
Frontend: React  
Backend: Node.js  
Database: Firebase  

## How to Run
npm install  
npm run dev
## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
